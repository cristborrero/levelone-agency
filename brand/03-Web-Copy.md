## Copy Guidelines (Quick Reference)

| Rule                | Application                                                               |
| ------------------- | ------------------------------------------------------------------------- |
| **Tone**            | Assertive, sleek, impactful — never corporate                             |
| **Sentence length** | Short. Punchy. Max 20 words per sentence.                                 |
| **Casing**          | Headings: ALL CAPS. Body: Sentence case. Labels: ALL CAPS.                |
| **Punctuation**     | Use em dashes (—) not hyphens. Full stops on statements, not on headings. |
| **Numbers**         | Use digits, not words. "50+" not "fifty-plus".                            |
| **Spelling**        | British English. "Optimised" not "Optimized". "Colour" not "Color".       |
| **Words to use**    | Build, craft, engineer, partner, sharp, lean, fast                        |
| **Words to avoid**  | Leverage, utilise, synergy, disrupt, best-in-class, game-changing         |
