## 1. Brand Name

### Name: **Levelone**

| Attribute           | Detail                                               |

| **Trading name**    | Levelone                                             |

### Name Rationale

- **"Level"** — evoca igualdad, equilibrio y estándar de calidad. Posiciona a la agencia como un socio que opera al mismo nivel que sus clientes.
- **"One"** — primera elección, excelencia singular, el punto de partida definitivo. Refuerza la noción de ser la agencia número uno.
- **Escrito junto** — tipográficamente moderno, pesa como una sola palabra-marca memorizable, ideal para branding digital y URL.

## 3. Brand Foundation

### 3.1 Brand Purpose

Existimos para cerrar la brecha entre la ambición digital y la ejecución de clase mundial, dando a empresas medianas del Reino Unido acceso al mismo calibre creativo y tecnológico que antes solo estaba disponible para las corporaciones.

### 3.2 Brand Vision

Ser la agencia digital de referencia para empresas medianas en el Reino Unido que exigen resultados premium sin la burocracia corporativa — un estudio donde la creatividad, la tecnología y la estrategia convergen sin fricciones.

### 3.3 Brand Mission

Diseñamos, desarrollamos y desplegamos experiencias digitales de alta fidelidad — webs, identidades visuales, contenido social, vídeo, audio, motion graphics e inteligencia artificial — con la obsesión por el detalle de un estudio boutique y la capacidad de entrega de una agencia a escala.

### 3.4 Brand Promise

Cada proyecto que sale de Levelone será técnicamente impecable, visualmente distintivo y estratégicamente alineado con los objetivos de negocio del cliente.

## 6. Brand Positioning

### 6.1 Target Audience

**Primary:** Empresas medianas del Reino Unido (£2M–£50M de facturación) con ambición de crecimiento digital, que actualmente trabajan con freelancers sueltos o agencias que no entregan calidad premium.

**Secondary:** Startups en fase de scale-up que necesitan una identidad digital sólida y escalable.

**Perfil del decisor:** Marketing directors, founders, COOs — personas que valoran la calidad, entienden de tecnología y buscan un partner, no un proveedor.

### 6.2 Competitive Differentiation

| Nosotros                             | Agencias tradicionales        | Freelancers             |
| ------------------------------------ | ----------------------------- | ----------------------- |
| Full-service con profundidad técnica | Full-service pero genéricos   | Especialización aislada |
| AI-native, procesos modernos         | Stack legacy, procesos lentos | Herramientas básicas    |
| Estética premium, dark editorial     | Corporativa, predecible       | Inconsistente           |
| Ownership total del resultado        | "No estaba en el scope"       | Disponibilidad variable |
| Velocidad de estudio boutique        | Burocracia de corporación     | Sin backup              |

### 6.3 Brand Pillars

1. **Craft** — Obsesión por la calidad de ejecución.
2. **Technology** — AI y herramientas de vanguardia como core del proceso.
3. **Strategy** — Cada decisión creativa sirve a un objetivo de negocio.
4. **Speed** — Entrega excepcional en tiempos excepcionales.
5. **Partnership** — Relación de socio, no de proveedor.

### 6.4 Positioning Statement

> Para empresas medianas del Reino Unido que merecen más de lo que su agencia actual les ofrece, **Levelone** es la agencia digital que fusiona diseño premium, tecnología de vanguardia e inteligencia artificial para crear experiencias digitales que impactan, convierten y escalan — con la intensidad de un estudio boutique y la capacidad de una agencia a escala.

## 7. Visual Identity System

### 7.1 Design Direction: Dark Editorial Brutalist

**Filosofía visual:** Espacio negativo generoso, tipografía con peso y escala extrema, contrastes duros, grid roto con intención, materialidad digital. La estética brutalista se suaviza con refinamiento editorial — el resultado es premium, no hostil.

### 7.2 Color System

```css
:root {
  /* ─── Primary Brand Colors ─── */
  --brand-black: #0a0a0a; /* Nero — fondo principal */
  --brand-white: #f5f0eb; /* Warm white — texto sobre oscuro */
  --brand-accent: #d4ff00; /* Electric lime — CTAs, highlights, poder */

  /* ─── Brand Black Scale ─── */
  --brand-black-deep: #050505; /* Headers hero */
  --brand-black-mid: #141414; /* Cards, superficies elevadas */
  --brand-black-soft: #1e1e1e; /* Bordes, separadores sutiles */

  /* ─── Neutral Greys ─── */
  --brand-grey-900: #2a2a2a; /* Texto secundario sobre oscuro */
  --brand-grey-700: #4a4a4a; /* Captions, metadata */
  --brand-grey-500: #6b6b6b; /* Placeholders */
  --brand-grey-300: #9a9a9a; /* Texto deshabilitado */
  --brand-grey-100: #e5e5e5; /* Bordes sobre claro */

  /* ─── Accent Variations ─── */
  --brand-accent-muted: #b8d94d; /* Hover states */
  --brand-accent-glow: rgba(212, 255, 0, 0.15); /* Glow effects */

```

| Swatch | Token            | Hex       | Uso                                        |
| ------ | ---------------- | --------- | ------------------------------------------ |
| ⬛     | `--brand-black`  | `#0A0A0A` | Fondo principal, hero sections             |
| ⬜     | `--brand-white`  | `#F5F0EB` | Texto principal, headings sobre oscuro     |

**Ratio de accesibilidad (WCAG AA):**

- `#F5F0EB` sobre `#0A0A0A` → **15.2:1** ✅
- `#D4FF00` sobre `#0A0A0A` → **14.8:1** ✅
- `#D4FF00` sobre `#F5F0EB` → **1.2:1** ❌ (nunca usar accent sobre blanco como texto)

### 7.5 Visual Elements

**Grid:** 12 columnas, con intencionados breaks de grid para efecto brutalista. Los elementos pueden sangrar fuera de su contenedor para crear tensión visual.

**Bordes:** 1px solid `--brand-grey-700` para separadores sutiles. Sin border-radius (0px) en componentes primarios — corners rectos = brutalismo. Excepción: avatares e inputs pueden usar `2px`.

**Imágenes:** Tratamiento duotone (negro + accent) para imágenes hero. Filtro `grayscale(100%)` con hover que revela color. Aspect ratios cinéticos: 21:9, 16:9.

**Efectos:**

- Hover: `translate` + cambio de color accent.
- Glow: `box-shadow: 0 0 40px var(--brand-accent-glow)` para CTAs principales.
- Cursor custom: crosshair o pointer personalizado en zones interactivas.
- Grain: overlay de textura noise al 3-5% de opacidad para materialidad.

## 10. Logo System _(especificaciones para diseño)_

### 10.1 Logo Concept

**Wordmark primary:** "LEVELONE" en Space Grotesk Black, todo uppercase, tracking tight. Sin símbolo — la tipografía ES la marca.

**Monogram:** "L1" ligature — para uso en favicon, avatares sociales y situaciones de espacio reducido.

### 10.2 Logo Variants

| Variante                     | Uso                               | Fondo                            |
| ---------------------------- | --------------------------------- | -------------------------------- |
| **Primary wordmark (light)** | Uso principal                     | Fondos oscuros (`--brand-black`) |
| **Primary wordmark (dark)**  | Impresión, fondos claros          | Fondos claros                    |
| **Monogram L1**              | Favicon, social avatars, app icon | Ambos                            |
| **Wordmark + tagline**       | Propuestas, documentos formales   | Ambos                            |

### 10.3 Clear Space & Minimum Size

- **Clear space:** La altura de la "L" alrededor de todo el logo.
- **Tamaño mínimo digital:** 120px de ancho (wordmark), 32px (monogram).
- **Tamaño mínimo impresión:** 35mm de ancho (wordmark), 10mm (monogram).

### 10.4 Logo Don'ts

- ❌ No rotar, distorsionar o estirar.
- ❌ No añadir sombras, gradientes o strokes.
- ❌ No colocar sobre fondos que comprometan la legibilidad.
- ❌ No cambiar la tipografía del wordmark.
- ❌ No usar el accent color como fondo del logo (legibilidad).

## 11. Brand Applications

### 11.1 Digital Touchpoints

| Aplicación            | Tratamiento                                                                                    |
| --------------------- | ---------------------------------------------------------------------------------------------- |
| **Website**           | Hero full-bleed dark, tipografía hero oversize, scroll animations, accent CTAs, grain overlay. |
| **Social media**      | Templates dark con accent highlights, tipografía bold, formato vertical 4:5 y 9:16.            |
| **Email signatures**  | Minimalista: wordmark + nombre + cargo en Inter, línea accent.                                 |
| **Proposals / Decks** | Fondo negro, tipografía blanca con accent highlights en datos y KPIs.                          |
| **Invoices**          | Clean, dark header con wordmark, cuerpo con fondo blanco para legibilidad.                     |

### 11.2 Motion & Animation Principles

- **Ease:** `cubic-bezier(0.25, 0.1, 0.25, 1)` — movimiento con peso, no robótico.
- **Duración:** 200-400ms para micro-interactions, 600-1000ms para transiciones de sección.
- **Dirección:** Bottom-to-top reveal como principio default.
- **Texto:** Máscara reveal por línea (stagger 80ms) para headlines hero.

## 12. Brand Protection

### 12.1 Trademark Strategy

- Registrar "LEVELONE" como wordmark en UK (UKIPO) clase 35 (advertising), 42 (software design).
- Registrar el monograma "L1" como marca figurativa.
- Asegurar dominios: `levelone.agency`, `leveloneagency.co.uk`, `leveloneagency.com`.

### 12.2 Usage Compliance

- Todos los assets de marca se distribuyen desde un repositorio centralizado (`/brand/assets/`).
- Cualquier uso externo del logo requiere archivo de guidelines incluido.
- Review trimestral de consistencia en todos los touchpoints digitales.

### 12.3 Brand Monitoring

- Google Alerts para "Levelone Agency" y variaciones.
- Monitoreo mensual de SERP para proteger positioning de marca.
- Auditoría anual de brand consistency score (target: 95%+).
